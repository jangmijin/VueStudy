Vue 라우터 Vue Routers -Vue를 이용한 SPA를 제작할 때 유용한 라우팅 라이브러리 라우터란? (라우트 화면이 전환할 때 전환하는 행위를 라우트라고 한다. Vue의 경우도 Vue를 이용한 SPA 싱글 페이지 어플리케이션 제작할 때 유용한 라우팅 라이브러리) SPA란? 일반적으로 웹페이지는 해당 웹페이지를 서버에 요청하고 서버가 html로 돌려줬을 때 그거를 브라우저 렌더링 엔진으로 뿌려내서 화면에 보여준다. 해당 화면에 대한 정보를 미리 가지고 있다가 해당 화면으로 넘어갈 때 서버에 요청하는 것이 아닌 내부적으로 라우터를 이용해서 서버에 요청해서 데이터를 받아오지 않고도 화면에 매끄럽게 전환을 해주는 게 싱글 페이지 어플리케이션으로 보면 된다. 따라서 뷰 뿐만 아니라 현대 모던 프레임워크들도 마찬가지로 다 지원을 하고 있다. -Vue코어 라이브러리 외에 Router라이브러리를 공식 지원하고 있고 아래와 같이 설치한다. npm install vue-router --save //es6기준 npm이나 yarn같은 패키지 인스톨러를 이용해서 설치 Vue 라우터는 기본적으로 RootUrl'/#/'{Router name}의 구조로 되어 있다. -example.com/#/user 여기서 #태그 값을 제외하고 기본 URL방식으로 요청 때마다 index.html을 받아 라우팅을 하려면
