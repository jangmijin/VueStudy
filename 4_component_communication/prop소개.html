props 소개 -모든 컴포넌트는 각 컴포넌트 자체의 스코프를 갖는다. 그렇기 때문에 다른 컴포넌트의 값에 바로 접근하지 못한다. 컴포넌트 통신에 대해 이해를 하고 props나 emit event, event bus에 대해서 알고 있어야한다 props를 왜써야할까!! ex) 하위 컴포넌트가 상위 컴포넌트의 값을 바로 참조할 수 없는 형식 -상위에서 하위로 값을 전달하려면 props 속성을 사용한다 props 설명 props 설명
